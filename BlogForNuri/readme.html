Активация виртуального окружения

Активация виртуального окружения производится командой: source env/bin/activate. Вы можете проверить что находитесь в вирт окружении командой: which python Он должен быть в env каталоге. Для того что бы покинуть вирт окружение введите: deactivate

Установка зависимостей

Для установки неоходимых пакетов воспользуйтесь командой: python3 -m pip install -r requirements.txt

Запуск проекта

Для запуска проекта введите в каталоге с файлом manage.py следующую команду: python manage.py runserver
