Создание виртуального окружения python -m venv env Активация виртуального окружения Активация виртуального окружения производится командой: env\Scripts\Activate.ps1 При исопльовании PowerShell env\Scripts\activate.bat При исопльовании cmd.exe Вы можете проверить что находитесь в вирт окружении командой: where.exe python Он должен быть в env каталоге. Для того что бы покинуть вирт окружение введите: deactivate Установка зависимостей Для установки неоходимых пакетов воспользуйтесь командой: python -m pip install -r requirements.txt Запуск проекта Для запуска проекта введите в каталоге с файлом manage.py следующую команду: python manage.py runserver
